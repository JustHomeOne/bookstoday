# Онлайн библиотека

Простая фронтенд-страница для управления библиотекой.

## Что есть

- `add.html` — отдельная страница добавления книги.
- `index.html` — главная страница со списком книг, поиском и фильтрами.
- `styles.css` — стили для удобного отображения на десктопе и мобильных устройствах.
- `script.js` — логика добавления книги и сохранения в `localStorage`.
- `index.js` — логика поиска, фильтрации и отображения списка книг.
- `library.html` / `library.js` — старые версии страницы со списком книг.

## Как пользоваться

1. Откройте `add.html` в браузере.
2. Заполните форму и нажмите «Добавить книгу».
3. Откройте `index.html`, чтобы посмотреть сохранённый список книг, использовать поиск и фильтрацию.
